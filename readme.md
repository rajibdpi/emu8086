# EMU8086 - 8086 Microprocessor Emulator

EMU8086 is a popular 16-bit microprocessor emulator and integrated assembler designed for the Intel 8086 CPU. It is widely used for educational purposes to help students learn Assembly language and understand the internal workings of a microprocessor.

## Key Features
- **Integrated Assembler**: Write and compile 8086 assembly code directly within the application.
- **Real-time Emulator**: Run and debug your programs step-by-step to visualize CPU behavior.
- **Virtual Hardware**: Includes simulations for virtual displays, stepper motors, and traffic lights.
- **Memory & Register Inspection**: Monitor changes in registers (AX, BX, CX, DX, etc.) and system memory in real-time.

## Getting Started
1. **Write Code**: Use the built-in editor to write your `.asm` source files.
2. **Compile**: Click the 'Compile' button to check for syntax errors and generate machine code.
3. **Emulate**: Click 'Emulate' to launch the emulator window.
4. **Run/Debug**: Use 'Single Step' to execute instructions one by one, or 'Run' to execute the program at full speed.

## System Requirements
- **Operating System**: Windows XP, 7, 8, 10, or 11.
- **Architecture**: Compatible with both 32-bit and 64-bit systems.
